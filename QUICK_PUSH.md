# Quick Push to GitHub - Copy & Paste Commands

## Fastest Method: Using Personal Access Token

### Step 1: Get Your Token
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Name: "Astrava"
4. Check: ✅ repo
5. Click "Generate token"
6. **COPY THE TOKEN** (starts with `ghp_`)

### Step 2: Push with Token
Open PowerShell in your project directory and run:

```powershell
# Replace YOUR_TOKEN_HERE with your actual token
git push https://YOUR_TOKEN_HERE@github.com/MrSloopyCoder/Astrava.git main
```

**Example:**
```powershell
git push https://ghp_abc123xyz789@github.com/MrSloopyCoder/Astrava.git main
```

### Step 3: Verify
Go to: https://github.com/MrSloopyCoder/Astrava

You should see all your files! 🎉
